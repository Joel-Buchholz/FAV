Herzklopfen

Ein Herz aus Zwei Pseudoelementen das durch die :Aktive Funktion zum schlagen gebracht wird.

.herz {
position: relative;
top: 5.75vh;
left: 2.2vw;
transform-origin: 25% 100%;
}

.herz:before, .herz:after {
position: absolute;
content: "";
left: 3.125rem;
top: 0;
width: 3.125rem;
height: 5rem;
background: red;
border-radius: 3.125rem 3.125rem 0 0;
transform: rotate(-45deg);
transform-origin: 0% 100%;
}

.herz:after {
left: 0;
transform: rotate(45deg);
transform-origin: 100% 100%;
}

.herz:active {
animation: pump 1s infinite;
}

@keyframes pump {
0% {
transform: scale(1);
}

25% {
transform: scale(0.9);
}

50% {
transform: scale(1.1);
}

75%{
transform: scale(0.9);
}

100% {
transform: scale(1);
}
}

Sprechblase

Eine Sprechblase aus einem Div und zwei Pseudoelementen in die durch die :Hover Funktion etwas reingeschrieben wird.

.sprechblase {
height: 5rem;
width: 8rem;
position: absolute;

border-radius: 1rem;
background: white;
top: 4vh;
left: 1.4vw;
}

.sprechblase:after{
content: '';
position: absolute;
border-style: solid;
border-width: 1.688rem 0.438rem 0;
border-color: white transparent;
width: 0;
z-index: 1;
bottom: -1.688rem;
left: 1.438rem;
}

.sprechblase:before{
content: '';
position: absolute;
border-style: solid;
border-width: 2.063rem 0.813rem 0;
border-color: black transparent;
width: 0;
z-index: 0;
bottom: -2.5rem;
left: 1.063rem;
}

.tippen {
width: 0;
font-size: 1.7rem;
overflow: hidden;
border-right: 0.25rem solid black;
white-space: nowrap;
margin: 0 auto;
animation: blink 0.75s step-end infinite;
}

.sprechblase:hover .tippen {
animation: tipp 1.5s steps(12,end) forwards,

}

@keyframes tipp {
from {
width: 0
}
to {
width: 100%
}
}

@keyframes blink {
from {
border-color: transparent;
}
50% {
border-color: black;
}
to {
border-color: transparent;
}
}

Pacman

Dieser Pacman besteht aus ein Div und zwei Pseudoelementen und wird durch die :Hover Funktion zum zuschnappen gebracht.

.pacman {
position: absolute;
top: 5.75vh;
left: 2.75vw;
border-right: 2.5rem solid transparent;
border-top: 2.5rem solid rgb(253, 228, 2);
border-left: 2.5rem solid rgb(253, 228, 2);
border-bottom: 2.5rem solid rgb(253, 228, 2);
border-radius: 100%;
}

.pacman:before,
.pacman:after {
content: "";
bottom: -4vh;
right: -2vw;
position: absolute;
border-right: 2.5rem solid transparent;
border-top: 2.5rem solid rgb(253, 228, 2);
border-left: 2.5rem solid rgb(253, 228, 2);
border-bottom: 2.5rem solid rgb(253, 228, 2);
border-radius: 100%;
}

.pacman:hover::before {
animation: schnapp 0.5s linear infinite;
}

.pacman:hover::after {
animation: schnapp2 0.5s linear infinite;
}

@keyframes schnapp {
from {
rotate: 0deg;
}
50% {
rotate: 45deg;
}
to {
rotate: 0deg;
}
}

@keyframes schnapp2 {
from {
rotate: 0deg;
}
50% {
rotate: -45deg;
}
to {
rotate: 0deg;
}
}
